Javascript

Automatically downloads episodes of a show on a user's "currently watching list" on Anilist. Made using 3 APIs:
one to retrieve the currently watching show, one to communicate with a website to download the episode and the last one to send that episode to the user's torrent application.

Language and programs used

  • Javascript
  • Nodejs
  • MariaDB
  • Qbittorent-nox
