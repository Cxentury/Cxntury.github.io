3D engine (WIP)

A 3D engine made from scratch using openGL, started as an open-ended school project and continued on my personal github. The models displayed here are the ones I made in Blender.
I was already interested in 3D graphics and this project helped me gain a deeper understanding of how it works behind the scenes (even though I still have a lot to learn).

Images from the engine


What I did

  • Managing the camera
  • Implementing all Shaders
  • Managing lighting, materials and textures.
  • Made quick post-processing effects like box blur and chromatic aberration
  • ...
The list is quite simplified, but all the source code can be found on my Github
A big part of what I learned is thanks to the website learnopengl.

Side Notes

This is still a huge work in progress as making a 3D engine is something really complex.
One problem is that blender cannot export anything other than Principled BSDF in .FBX,
so I had to reduce all my textures to basic BSDF which looks quite unnatural.
